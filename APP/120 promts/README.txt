# JobBoost Prompt Kit

A practical prompt pack for job seekers who want faster, higher-quality applications using any AI assistant.

## What you get
- `STARTER_PROMPTS.md` — 60 high-utility prompts
- `PROMPTS.md` — full 120-prompt pack
- `CATEGORY_CHEATSHEETS.md` — what to prepare for each category
- `TEMPLATES_STANDARD_DOCS.md` — blank standardized forms (EU CV, US Resume, cover letter, interview, outreach, negotiation)
- `pdf-templates/` — ready blank PDF files you can upload to AI as structure references
- `LISTING_COPY.md`, `FAQ.md`, `LAUNCH_COPY.md` — sales and launch materials

## Quick start (works with ChatGPT, Claude, Gemini, Copilot, etc.)
1. Open `TEMPLATES_STANDARD_DOCS.md`.
2. Copy one blank template (example: EU CV or US Resume) and fill it with your data.
3. Open `CATEGORY_CHEATSHEETS.md` and collect the minimum inputs for that category.
4. Open `STARTER_PROMPTS.md` (or `PROMPTS.md`) and pick a prompt.
5. Replace every **bold bracket field** with your real data.
6. Paste the prompt into your AI tool.
7. Ask for 2-3 variants (safe / balanced / bold), then choose one.
8. Run one final polish prompt: clarity + grammar + ATS readability.

## How to build a finished document in any AI (step-by-step)
### A) Build a CV/Resume
1. Fill a source template from `TEMPLATES_STANDARD_DOCS.md`:
   - EU CV template OR US Resume template.
2. Paste that draft into a CV prompt from the kit.
3. Add the target job description in the prompt.
4. Ask AI for:
   - keyword alignment,
   - quantified bullet points,
   - 1-page final version.
5. Export final version into your editor (Docs/Word/Canva).

### B) Build a tailored Cover Letter
1. Fill the Cover Letter Input Sheet in `TEMPLATES_STANDARD_DOCS.md`.
2. Use a Cover Letter prompt from the kit.
3. Include company info + role requirements.
4. Request two tones (formal + modern) and pick one.

### C) Build LinkedIn profile text
1. Fill LinkedIn Profile Worksheet template.
2. Use LinkedIn prompts (headline, about, experience, outreach messages).
3. Publish best version and keep one variant for A/B testing.

### D) Build Interview answers
1. Fill Interview Prep Worksheet.
2. Use Interview prompts for STAR answers and mock interview.
3. Ask AI to score each answer 1-10 and improve weak parts.

### E) Build outreach and follow-up system
1. Fill Outreach Tracker template.
2. Use outreach/follow-up prompts.
3. Keep message variants for recruiter, hiring manager, and referral asks.

### F) Build offer negotiation package
1. Fill Offer Comparison + Negotiation Worksheet.
2. Use negotiation prompts to produce email/script variants.
3. Choose your preferred package (salary vs benefits vs flexibility).

## Placeholder rule
All required inputs in prompts are written as **bold bracket fields**.
Example: **[enter your target job title here]**

Never leave bracket fields empty. If you don’t have one value, write `N/A` and continue.
