# Oscar Pick Site (GitHub-ready)

Sve je pripremljeno u ovom folderu za upload na GitHub:

- `index.html` — cijela stranica
- `assets/oscar-banner.png` — tvoja slika (dodaj ručno)

## Što trebaš napraviti

1. Kopiraj svoju sliku u:
   - `oscar-pick-site/assets/oscar-banner.png`
2. Pushaj folder na GitHub.
3. Ako koristiš GitHub Pages, stavi root na ovaj folder ili prebaci sadržaj foldera u root repozitorija.

Napomena: stranica već ima:
- sve Oscar 2025 kategorije i nominacije
- QR kod po nominaciji
- copy pay link / copy memo
- CSV export (setup + payout template)
